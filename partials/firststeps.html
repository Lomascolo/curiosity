First step

Main page overview

As you can see on the previous screenshot, curiosity is divided in three main area :

  • The First area allows you to add/choose an elasticsearch server and to select an alias or index to work on.
  • The second area is where you will work, you can create and personalize your layout, add modules, navigate through workspaces ...
  • And then the last area is the toolbar, that give you the possibility to save your work, launch query, open menus ...
Let's create our first workspace and launch some queries.

Step 1 : Select your server and index

On the first area, there are two drop down list(1) :

  • The first one allows you to choose a server. If you use curiosity for the first time, normally there is only one server in this list. If you want to add a new server, press "New" button, fill the input that just appeared, then click on connect. If you curiosity was able to reach the server, "connect" button stays green (2) and your server is added to the server list, otherwise the button goes red (3).
  • The second list contains all current server indexes.

Choose a server and an index then let's customize your layout

The green/red light represents the curiosity's status. If it goes red, an error has occurred. If you want more information about it, check the developper console (a curiosity's console will soon be added).

Step 2 : Configure your layout

Now you have chosen your server and index, it's time to create your first workspace and customize its layout.

First step, give a name to your workspace (1) (not required), then, you have to choose the number of line and columns of your workspace. You can do this by filling corresponding inputs (2,3) or through the square (4) by clicking on the layout you want. Then click on "ok" (4).

Your layout is created, let's add some modules.

Step 3 : Add modules

Curiosity works with modules, each module has differing goals, there are modules to display results, built queries, make aggregations... Each workspace is divide in cards that are the representation of your layout. In each card you can add as many module as you want.

When you want to add a module to a card :

  • Click on the well on the top right corner of the card (1).
  • A window will appear, select a module in the module list (2), click on "add module", the the module will be added in the card's temporary module list (3).For this example add "Query" and "result" modules to your card.
  • Then press "ok"(6) to add your module in your card.

If you want to add some styles to your card, fill the Card's style (5) input with css styles.

Step 4 : Launch a query

Curiosity use elasticsearch's query string query syntax, so you can easily create complex queries. Click in the query form (first form in page center), enter your query then press "enter" or click on search button. If your request is syntaxly correct, the status light remains green and the result will appear at the bottom center of the page.

Here are some basics for building a query :

  • If you want to search a string on a specific field : 
    the.path.to.my.fields:"string you search"
  • Getting docs that matchs a value range for a specific field : 
    the.path.to.my.fields:[Start TO End]
